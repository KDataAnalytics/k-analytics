# Microsoft Copilot for Analysts

Real-world examples of using **Microsoft 365 Copilot** in Excel and PowerPoint to turn raw data into executive-ready insights and presentations — in seconds.

### What It Shows
- Upload CSV → Copilot instantly analyzes trends, anomalies, and gaps
- Ask natural language questions → get charts, summaries, and full slide decks
- Zero code — 100% native M365 Copilot (2025)

### Files
- `sales_with_forecast.xlsx` → Original data
- `sales_trends.pptx` → Fully AI-generated deck (slides, charts, insights)

### Live Example
Ask Copilot in Excel: “Compare actual vs forecast sales” → watch it build the analysis and PowerPoint automatically.
